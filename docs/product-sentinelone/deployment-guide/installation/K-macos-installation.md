# macOS 에이전트 설치 
---
> <br />Apple은 개발자에게 출시 전 GA(일반 사용자 출시) 버전을 제공하지 않으며, 베타 버전(개발자와 공유됨)과 GA 버전 사이에 변경 사항이 있을 수 있습니다. <br />SentinelOne이 새로운 macOS 버전을 공식적으로 지원할 때까지 OS 업데이트를 중단할 것을 권장합니다.<br /> Apple의 새로운 OS 및 하드웨어 출시에는 특정 요구 사항이 있을 수 있습니다. 이러한 출시로 인해 SentinelOne 솔루션의 특정 기능과 기능성에 영향을 미칠 수 있습니다.<br /><br />

## 2. macOS 하드웨어 요구 사항

## 3. macOS SentinelOne 에이전트 설치

### 1) 에이전트 설치 파일 다운로드 

- **➊ Sentinels** > **➋ Packages** > **➌ All Platform Types** > **➍ MacOS**<br />![Agent Version List](./img/macOS/package_number.png)<br />
![Agent Version List](./img/macOS/package_down.png)
- 테스트 및 안정화가 완료된 **GA 버전**으로 다운로드 해주세요.
- EA 버전은 테스트 전용 에이전트이며 예상치 못한 버그나 문제를 일으킬 수 있습니다.
<br />

### 2) 설치 토큰 값 확인
- **➊ SENTINELS** > **➋ PACKAGES** > **➌ Site Token**<br />![alt text](./img/windows/image-17.png)
   - Site Token 정보를 확인하려면 **최소 Site Scope**을 선택해야 합니다. <br />
   - ex) Global / 00_(KR)_PAGO Account / **Installation**
<br />

### 3) macOS용 SentinelOne 에이전트 설치
- 설치를 시작하고 토큰 값을 입력하세요.<br />![Agent Enter Token](./img/macOS/enter_token.png)<br />

&nbsp;
- SentinelOne Extensions가 네트워크 확장을 사용하도록 허용하세요.<br />![Agent Network Extension](./img/macOS/network_extension.png)<br />

&nbsp;
- 설치가 성공적으로 완료되었는지 확인하세요.<br />![Agent Installaion Successfully Completed](./img/macOS/installaion_complete.png)<br />

&nbsp;
- 시스템 설정 > 일반 > 로그인 항목 > 확장 프로그램 경로를 따라가세요.<br />![Agent System Setting](./img/macOS/system_setting1.png)<br />

&nbsp;
- 네트워크 확장 기능을 선택하세요.<br />![Agent System Setting](./img/macOS/system_setting2.png)<br />

&nbsp;
- SentinelOne Extensions 항목이 활성화되어 있는지 확인하고, 그렇지 않다면 활성화하세요.<br />![Agent Ssytem Setting](./img/macOS/system_setting3.png)<br />

&nbsp;
- 시스템 설정 > 개인정보 보호 및 보안 > 전체 디스크 접근 권한 경로를 따라가세요.<br />![Agent System Setting](./img/macOS/system_setting4.png)<br />

&nbsp;
- Sentinel, Sentinel_helper, 및 Sentinel_shell 항목에 대해 전체 디스크 접근 권한을 허용하세요.<br />![Agent System Setting](./img/macOS/system_setting5.png)<br />

&nbsp;
- 설치된 SentinelOne UI를 확인하고 우측 상단에서 온라인 상태를 확인하세요.<br />![Agent UI](./img/macOS/agent_UI.png)
