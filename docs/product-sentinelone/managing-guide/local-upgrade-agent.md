# 로컬에서 SentinelOne Windows 업그레이드하는 방법
---
<br />
:::tip Note
- 일부 Windows OS의 경우 **SentinelInstaller_windows_64bit_v23_4_6_347.msi** 버전 Agent 설치파일이 지원하는 최신 릴리즈 입니다.<br />**➔ Windows Server 2008 R2, 2012, Windows 7**
- 그 외 대부분의 Windows OS는 콘솔 상의 가장 최신 **GA**버전 Agent 설치파일을 다운로드 받아 진행하십시오.
:::
<br />

## 1. Agent 설치 파일 실행 
- Agent 설치파일 실행 > **[Upgrade]** 클릭
![alt text](./img/upgrade/Screenshot%202025-03-20%20at%2014.55.06.png)
<br />

## 2. Agent 업그레이드
![alt text](./img/upgrade/Screenshot%202025-03-20%20at%2014.55.50.png)
<br />

## 3. 업그레이드 완료 확인
- **[Finish]** 클릭 <br />![alt text](./img/upgrade/Screenshot%202025-03-20%20at%2014.56.21.png) 
<br />

## 4. UI에서 업그레이드 확인
- UI에서 Agent Details ➔ Version 업그레이드 확인
