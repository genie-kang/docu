### 1) 에이전트 설치 파일 다운로드
- **➊Settings**  > **➋ Deployments** > **➌ Installers[Protect 선택]** > **➍ Download**<br />![alt text](./img/linux/image.png)<br />

&nbsp;
### 2) 설치 토큰 값 확인
- **➊Settings**  > **➋ Application** > **➌ Installation Token**<br />![alt text](./img/linux/image-2.png)<br />

&nbsp;
### 3) MSI 파일 설치
- **➊ CylacnePROTECT.msi** > **➋ 마우스 우클릭** > **➌ 설치** <br />![alt text](./img/windows/image-1.png)<br />

&nbsp;
- **➊ Site Token 입력** > **➋ Next**<br />![alt text](./img/windows/image-2.png)

&nbsp;
- **Finish**클릭<br />![alt text](./img/windows/image-4.png)

&nbsp;
### 4) 에이전트 설치 상태 확인
- 오른쪽 하단의 **CylancePROTECT 에이전트 트레이 아이콘** 확인<br />![alt text](./img/windows/image-5.png)<br />
&nbsp;
- **CylancePROTECT 아이콘** 우클릭 > **소개** 클릭<br />![alt text](./img/windows/image-6.png)<br />
&nbsp;
- 소개 팝업창에서 정보 확인
   - 정책
   - Online 상태
   - 폴리시의 최종 갱신 일시
   - 일련번호<br />
![alt text](./img/windows/image-7.png)
