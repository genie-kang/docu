# 오프라인 에이전트 및 통신 문제 해결
---

## 1. 443 포트 통신 여부 확인

443 포트가 방화벽에서 열려 있는지 확인하기 위해 Telnet 명령어로 통신 가능 여부를 테스트합니다.
- `telnet <your_console.cylance.com> 443` 명령어를 실행하여 443 포트를 통해 에이전트와 콘솔 간 통신 확인하십시오.<br />
  - telnet api.cylance.com 443
  - telnet data-au.cylance.com 443
  - telnet update-au.cylance.com 443
  - telnet protect-au.cylance.com 443
  - telnet download.cylance.com 443
  - telnet Venueapi-au.cylance.com 443<br /> 
  
**참고사항 : telnet client 미설치 시 설치가 필요합니다.**
